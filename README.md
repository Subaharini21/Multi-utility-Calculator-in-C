# Multi Utility Calculator in C

This project implements a **Multi Utility Calculator in C** with the following features:

✅ Simple Calculator (`+`, `-`, `*`, `/`, `%`)  
✅ Quadratic Equation Solver  
✅ Matrix Addition  
✅ Scientific Calculator (`sin`, `cos`, `tan`, `sqrt`, `power`)

It is designed for **beginner C practice, mini projects, and learning structured programming**.

---

## 💻 Features

1️⃣ Simple Calculator
Performs basic arithmetic operations on two numbers.

2️⃣ Quadratic Equation Solver
Calculates real or imaginary roots of quadratic equations.

3️⃣ Matrix Addition
Takes two matrices as input and displays their sum.

4️⃣ Scientific Calculator
Performs:
- `sin`, `coss`, `tan` (input in degrees)
- `sqrt`
- `power`
